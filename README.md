# Dev Snippets Manager

A modern web application for developers to save, organize, search, and manage code snippets easily.

---

## Features

- Add snippets
- Edit snippets
- Delete snippets
- Favorite snippets
- Search snippets
- Dark mode
- Local storage persistence
- Responsive design

---

## Tech Stack

- React
- Vite
- CSS
- TailwindCSS

---

## Installation

```bash
npm install
npm run dev
```
